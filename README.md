[![SQL Server](https://img.shields.io/badge/SQL_Server-2022-red.svg)](https://www.microsoft.com/sql-server)

Este proyecto es una implementación práctica desarrollada siguiendo el curso [**.NET Backend: .NET Core, SQL Server y seguridad JWT**](https://cursos.devtalles.com/courses/NET-Backend) de DevTalles.

## 🎯 Propósito del Proyecto
El objetivo principal de este repositorio es el aprendizaje y dominio de las bases para construir aplicaciones API robustas utilizando **C#** en **.NET 8**, enfocándose en la arquitectura de backend, persistencia de datos y seguridad.

## 🚀 Tecnologías y Herramientas
* **Framework:** .NET 8 (ASP.NET Core Web API)
* **Lenguaje:** C#
* **Base de Datos:** SQL Server
* **Seguridad:** Autenticación y Autorización mediante JSON Web Tokens (JWT)
* **ORM:** Entity Framework Core

## 🧠 Conceptos Implementados
* Configuración de servicios e Inyección de Dependencias.
* Creación y gestión de Endpoints RESTful.
* Integración de seguridad JWT para protección de rutas.
* Modelado de datos y migraciones con SQL Server.
